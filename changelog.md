# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

See the full changelog at https://github.com/iamkaf/mochila2

## 4.1.0

### Added

- Quick stash support for all copper chest variants.
- Keybind (V) to open the ender backpack.

### Fixed

- Ender backpack shows in the creative tab.

## 4.0.0

### Changed

- Ported to Minecraft 1.21.11.
- Pausing Minecraft Forge support until it supports Gradle 9.

## 3.0.0

### Added

- Added Forge support

### Changed

- Ported to Minecraft 1.21.10

### Removed

- Mochila no longer requires Architectury API

## 2.0.0

### Added

## 1.4.0

### Added

- Second mode for the quick stash feature: “store” and “dump”.

    * **Store**: stores items already in the chest.
    * **Dump**: stores everything.

### Fixed

- Colored shulker boxes can no longer be placed inside backpacks.
- Blacklisted items can no longer be stored in backpacks via the swap offhand hotkey.

## 1.3.2

### Fixed

- Added missing criteria for mod recipes.

## 1.3.1

### Changed

## Types of changes

- `Added` for new features.
